# Beautyscraper #


### A Ziptask project for scraping beauty sites. ###

Scrape 10 sites as listed by the client.


### How do I get set up? ###

* ` pip install requirements.txt`
* `scrapy crawl <site_name> `

### List of sites to crawl ###

* beautybay.com
* maccosmetics.com
* cultbeauty.co.uk
* sephora.com
* maybelline.co.uk
* selfridges.com
* Polyvore.com
* net-a-porter.com
* shopstyle.co.uk
* beautylish.com
